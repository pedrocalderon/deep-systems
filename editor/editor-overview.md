# Editor Overview

This project has the objective to implement a funcional text editor from scratch
in Rust.

My objectives in doing so are not to replace my Neovim setup, but merrily to
study a subject that I like (text editors) as a project to ground my studies in
systems ans tools engineering and to learn/practice Rust.

I'm in no hurry to implement this, it will be implemented along side my studies
described in the [[curricula]].
