# BLOCK 3 — Algorithms & Data Structures for Tools

## Objective

Develop deep mastery of practical data structures:

- Balanced trees
- Gap buffer
- Rope
- Piece table
- LRU cache
- Diff algorithms
- Incremental data structures

Learn amortized analysis informally but rigorously.

## Prerequisites

- Basic asymptotics
- Block 1

## Resources

- _Open Data Structures_
- Selected CLRS chapters
- Roughgarden lectures (algorithm paradigms)

## Local Deliverables

- Implement:
  - Gap buffer
  - Rope
  - Piece table

- Benchmark comparison suite
- Implement Myers diff algorithm
- Implement LRU cache

## Text Editor Deliverables

- Replace naïve buffer with gap buffer
- Later swap to rope or piece table
- Add undo/redo model
- Add diff-based change tracking
- Add incremental line index structure
